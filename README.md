# Java Buzz – Specialty Coffee Shop

Java Buzz is a modern, specialty coffee shop located in the heart of Manhattan. This website showcases our premium coffee, seasonal menu, and inviting location.

## Features
- **Dynamic Menu**: Explore our full range of coffees, teas, and pastries with real-time pricing.
- **Interactive Location**: Find us easily via our Leaflet-powered map, with integrated directions and parking information.
- **Java Buzz Blog**: Stay updated with our latest brewing tips, sustainability reports, and community news.
- **Online Ordering**: Place your order online for a seamless pick-up experience.
- **Responsive Design**: Optimized for a premium experience on desktop, tablet, and mobile.
