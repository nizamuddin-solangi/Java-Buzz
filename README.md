# Java Buzz – Specialty Coffee Shop

Java Buzz is a modern, premium web experience designed for a specialty coffee shop in the heart of Manhattan. The website combines high-end aesthetics with advanced functionality, providing customers with a seamless way to explore the menu, find the shop, and stay updated with the latest coffee culture.

## 🚀 Live Features

### 🌙 Espresso Dark Mode
- **Persistent Theme**: A high-contrast dark mode palette ("Espresso Dark") that persists across page reloads and navigation sessions using `localStorage`.
- **Dynamic Toggle**: A smooth, animated theme switcher integrated directly into the navigation bar.

### ❤️ Personalized Favorites System
- **Curated Menu**: Users can "Heart" any item on the menu to save it to their local browser session.
- **Smart Filtering**: A dedicated "Favorites" tab in the menu allows users to instantly access their preferred drinks and snacks.

### 🗺️ Interactive Location & Map
- **Leaflet.js Integration**: A custom-styled interactive map featuring the shop's location.
- **Local Context**: Map markers for nearby public parking, subway stations, and landmarks.
- **Detailed Directions**: Comprehensive instruction cards for travel by subway, car, or foot, plus integrated Google Maps direction links.

### 📰 Dynamic JSON Blog
- **Data-Driven Content**: 12+ blog posts populated dynamically from JSON data.
- **Category Filtering & Search**: Robust search and filtering system to find brewing tips, sustainability reports, and news.
- **Social Sharing**: Built-in platform-specific sharing for **X (Twitter)** and **Facebook** to satisfy shareability requirements.

### 🥐 Smart Menu System
- **Real-time Filtering**: Categorized tabs for Coffee, Tea, Pastries, and Snacks.
- **Visual Feedback**: Hover animations and "Popular" badges for signature items.

## 🎨 Design & User Experience
- **Micro-animations**: Custom "Coffee Steam" animation for the logo and floating icons for a lively, interactive feel.
- **Premium Aesthetics**: Warm brown and forest green color scheme with glassmorphism effects and refined typography.
- **Fully Responsive**: Optimized for professional display on Desktop, Tablet, and Mobile devices (verified across multiple breakpoints).
- **Custom 404 Page**: A branded "Empty Cup" error page to maintain the premium feel throughout the user journey.

## 🔍 SEO & Technical Optimization
- **Meta Tags**: Optimized titles, keywords, and descriptions on every page.
- **Sitemap**: A comprehensive `sitemap.xml` for search engine crawlers.
- **Robots Policy**: Both `robots.txt` and `robot.text` included to match literal project requirements.
- **Performance**: High-quality imagery with optimized CSS/JS delivery.

## 📂 Project Structure
- `index.html`: Hero banner, signature creations, and testimonials.
- `menu.html`: Dynamic product grid with favorites and filtering.
- `location.html`: Leaflet map and travel directions.
- `blog.html`: Dynamic article system.
- `404.html`: Custom error handling page.
- `data/`: JSON storage for blog and menu content.
- `js/`: Modular logic for cart, map, theme switching, and data rendering.
- `css/`: Specialized stylesheets for main design and responsiveness.
